API

Referencia completa de la interfaz pública del componente.

Constructor

new Search(params: SearchParams): Search

Crea una nueva instancia del componente. Ver Configuración para los parámetros disponibles.

Métodos públicos

init()

init(): Search

Inicializa el componente, renderiza la estructura DOM y configura los event listeners. Debe llamarse después del constructor.

const search = new Search({ element: '#app', data: items }).init();

draw(searchTerm?, isEvent?)

draw(searchTerm?: string, isEvent?: boolean): Promise<Search>

Ejecuta una búsqueda y renderiza los resultados. Se llama automáticamente al escribir en el input. Incluye protección contra condiciones de carrera: si se ejecuta una nueva búsqueda antes de que termine la anterior, la anterior se descarta automáticamente.

Parámetro Tipo Default Descripción
searchTerm string this.searchTerm Término de búsqueda
isEvent boolean false Si fue iniciado por evento del usuario

sort(field, order?)

sort(field: string, order?: 'asc' | 'desc'): Search

Ordena los datos por un campo específico. En modo servidor, ejecuta una nueva búsqueda.

search.sort('nombre', 'asc');
search.sort('precio', 'desc');

clearSort()

clearSort(): Search

Elimina el orden actual y reinicia al orden natural de los datos.

clear()

clear(): Search

Resetea el estado del componente sin destruirlo. Limpia el término de búsqueda, el input, los resultados, la paginación, el caché y la selección actual.

search.clear(); // Resetea todo el estado

on(eventName, callback)

on<K extends keyof SearchEventMap>(event: K, callback: (data: SearchEventMap[K]) => void): EventEmitter<SearchEventMap>

Registra un listener para un evento. El tipo del dato en el callback se infiere automáticamente desde SearchEventMap. Retorna el EventEmitter para encadenamiento.

search
    .on('search', (data) => console.log('Búsqueda:', data))
    .on('pageChange', (data) => console.log('Página:', data.page))
    .on('itemSelected', (data) => {
        console.log('Seleccionado:', data.item);
        data.close(); // Cierra resultados
    });

getCacheKey(searchTerm, page)

getCacheKey(searchTerm: string, page: number): string

Genera la clave de caché para una búsqueda y página específicas.

setupKeyboardNavigation()

setupKeyboardNavigation(): Search

Configura la navegación por teclado (flechas arriba/abajo, Enter para seleccionar).

destroy()

destroy(): void

Destruye la instancia, limpiando todos los event listeners, observer de scroll y referencias internas. Las propiedades se re-inicializan en lugar de nullificarse para evitar errores de referencia nula. También llama a renderer.destroy() para limpiar timeouts y referencias del DOM.

search.destroy(); // Limpia todo

Métodos del Renderer

renderer.updateCounter()

renderer.updateCounter(data: { from: number; to: number; total: number; textPagination?: string }): void

Actualiza el contador de registros con formato personalizado.

search.renderer.updateCounter({
    from: 1,
    to: 10,
    total: 100,
    textPagination: '{{from}} - {{to}} de {{total}}'
});

renderer.showResults() / hideResults() / toggleResults()

renderer.showResults(): void
renderer.hideResults(): void
renderer.toggleResults(): void
renderer.hideResultsWithDelay(delay?: number): void

Controla la visibilidad del panel de resultados.

search.renderer.showResults();
search.renderer.hideResults();
search.renderer.toggleResults();
search.renderer.hideResultsWithDelay(300); // Con delay personalizado

renderer.destroy()

renderer.destroy(): void

Limpia timeouts de animación, hideTimeout y reinicia las referencias del DOM. Se llama automáticamente desde search.destroy().

Métodos de Paginación

pagination.getRange()

pagination.getRange(): { from: number; to: number; total: number }

Retorna el rango actual de items visibles para el contador.

const range = search.pagination.getRange(); // { from: 1, to: 10, total: 100 }

pagination.getTotalPages()

pagination.getTotalPages(): number

Retorna el total de páginas disponibles.

pagination.getTotalLoaded()

pagination.getTotalLoaded(): number

Retorna el total de items cargados (considerando scroll infinito).

pagination.loadNextPage()

pagination.loadNextPage(): number

Avanza a la siguiente página en modo scroll infinito.

pagination.hasMorePages()

pagination.hasMorePages(): boolean

Verifica si hay más páginas disponibles.

Encadenamiento

La mayoría de métodos retornan this para permitir encadenamiento:

search
    .sort('nombre', 'asc')
    .draw('laptop')
    .then(() => console.log('Búsqueda completada'));
