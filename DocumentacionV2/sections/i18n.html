Internacionalización

El componente incluye soporte integrado para múltiples idiomas mediante un sistema de traducciones configurable.

Idioma por defecto

Por defecto, el componente usa español. Las traducciones predeterminadas son:

const DEFAULT_TRANSLATIONS = {
    searchPlaceholder: 'Ingrese palabra clave...',
    searchLabel: 'Filtrar por Búsqueda',
    noResults: 'No se encontraron resultados',
    loading: 'Buscando...',
    pagination: '{{to}} de {{total}}'
};

Personalizar traducciones

Pasa un objeto translation al constructor para reemplazar las cadenas:

const search = new BuscadorDinamico.Search({
    element: '#app',
    data: items,
    translation: {
        searchPlaceholder: 'Search...',
        searchLabel: 'Search field',
        noResults: 'No results found',
        loading: 'Loading...'
    }
}).init();

Soporte multiidioma completo

Ejemplo para múltiples idiomas:

const translations = {
    es: {
        searchPlaceholder: 'Escriba para buscar...',
        searchLabel: 'Campo de búsqueda',
        noResults: 'No se encontraron resultados',
        loading: 'Cargando...'
    },
    en: {
        searchPlaceholder: 'Type to search...',
        searchLabel: 'Search field',
        noResults: 'No results found',
        loading: 'Loading...'
    },
    pt: {
        searchPlaceholder: 'Digite para pesquisar...',
        searchLabel: 'Campo de pesquisa',
        noResults: 'Nenhum resultado encontrado',
        loading: 'Carregando...'
    }
};

// Seleccionar idioma dinámicamente
const userLang = navigator.language.substring(0, 2);
const search = new BuscadorDinamico.Search({
    element: '#app',
    data: items,
    translation: translations[userLang] || translations.es
}).init();

Referencia de claves

Clave Descripción Ubicación
searchPlaceholder Texto placeholder del input de búsqueda Input del buscador
searchLabel Label accesible del input (aria-label) Input del buscador
noResults Mensaje cuando no hay resultados Panel de resultados
loading Mensaje de carga Indicador de carga
pagination Template del contador: {{from}}, {{to}}, {{total}} Contador de registros

La clave pagination acepta un template con las variables {{from}}, {{to}} y {{total}}:

translation: {
    pagination: '{{from}} - {{to}} de {{total}}'  // "1 - 10 de 100"
}

Cambiar idioma dinámicamente

// Para cambiar las traducciones después de la inicialización,
// necesitas crear una nueva instancia:
function changeLanguage(lang) {
    // Destruir la instancia anterior
    search.destroy();

    // Crear nueva instancia con el nuevo idioma
    search = new BuscadorDinamico.Search({
        element: '#app',
        data: items,
        translation: translations[lang]
    }).init();
}
💡
Consejo

Las traducciones se aplican solo al momento de la inicialización. Para cambiar el idioma en tiempo de ejecución, debes destruir y recrear la instancia.
