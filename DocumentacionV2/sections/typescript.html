TypeScript

El componente está escrito en TypeScript e incluye tipos completos para una experiencia de desarrollo segura.

Tipos exportados

Interfaz SearchParams

interface SearchParams {
    element: string;
    data?: Record<string, any>[];
    theme?: string;
    itemsPerPage?: number;
    debounceTime?: number;
    keyboardEnabled?: boolean;
    cacheEnabled?: boolean;
    cacheMaxSize?: number;
    cacheTtlSeconds?: number;
    zIndex?: number;
    dom?: string;
    sortBy?: string | null;
    sortOrder?: 'asc' | 'desc';
    template?: string | Function | null;
    translation?: Partial<TranslationCache>
    procesServer?: boolean;
    fetch?: FetchConfig;
    developmentMode?: boolean;
}

Interfaz FetchConfig

interface FetchConfig {
    url: string;
    method?: 'GET' | 'POST' | 'PUT' | 'DELETE';
    headers?: Record<string, string>;
    body?: Record<string, any>;
    timeout?: number;
}

Eventos tipados

interface SearchEventInit {

}

interface PageChangeEventData {
    page: number;

interface SortChangeEventData {
    field: string;

}

interface ItemSelectedEventData {
    item: Record<string, any>;

}

interface ItemHighlightedEventData {
    item: Record<string, any>;

interface DestroyEventData {

}

Uso con tipos

import { Search, SearchParams } from 'buscador-dinamico';

// Definir tipos para tus datos
interface Producto {
    id: number;
    nombre: string;
    precio: number;
    categoria: string;
}

const productos: Producto[] = [
    { id: 1, nombre: 'Laptop', precio: 999, categoria: 'Electrónica' }
];

// Configuración tipada
const config: SearchParams = {
    element: '#app',
    data: productos,
    theme: 'clean-white',
    template: (item: Producto) => `
        <div>
            <h3>${item.nombre}</h3>
            <p>$${item.precio}</p>
        </div>
    `
};

const search = new Search(config).init();

// Eventos tipados
search.on('itemSelected', (data: ItemSelectedEventData) => {
    const producto = data.item as Producto;
    console.log(producto.nombre);
});

Generación de tipos

Los archivos de tipos se generan automáticamente durante el build:

dist/
├── buscador-dinamico.es.js
├── buscador-dinamico.umd.js
├── buscador-dinamico.d.ts        ← Tipos principales
├── buscador-dinamico.d.ts.map    ← Source map de tipos
└── css/
    └── buscador-dinamico.css

Type narrowing

search.on('itemSelected', (data) => {
    // TypeScript sabe que data.item es Record<string, any>
    // Puedes hacer type assertion si conoces la estructura
    const item = data.item as Producto;
    console.log(item.nombre); // OK
});
💡
tsconfig.json

Asegúrate de que tu tsconfig.json incluya "moduleResolution": "node" o "moduleResolution": "bundler" para resolver correctamente los tipos.
