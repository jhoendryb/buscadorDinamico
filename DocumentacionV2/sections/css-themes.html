Temas CSS

El componente incluye 5 temas CSS predefinidos y un sistema completo de variables personalizables.

Temas disponibles

Tema Descripción Ideal para
adaptative Se adapta al tema del sistema (light/dark) Cualquier sitio web
clean-white Diseño limpio y minimalista en blanco Sitios corporativos
blue-black Tema oscuro con acentos azules Paneles de administración
onyx-black Tema completamente oscuro Aplicaciones nocturnas
forest-green Tema oscuro con tonos verdes Sitios de naturaleza

Aplicar un tema

const search = new BuscadorDinamico.Search({
    element: '#app',
    data: items,
    theme: 'onyx-black'
}).init();

Cambiar tema dinámicamente

// Cambiar tema después de la inicialización
search.renderer.setTheme('forest-green');

    --search-width: 280px;

    /* Espaciado entre elementos */

Resaltado

.app-search {
    /* Color de fondo del texto resaltado */
    --search-highlight-bg-color: #ffeb3b;

    /* Color del texto resaltado */
    --search-highlight-text-color: #000;
}

El resaltado se aplica usando la clase .search-highlight en los elementos <span> generados por el componente. Puedes personalizar estos estilos o agregar una clase adicional con el parámetro highlightClass:

/* Tema personalizado con resaltado naranja */
.app-search.theme-custom .search-highlight {
    background-color: #ff9800;
    color: #fff;
    border-radius: 3px;
    padding: 0 3px;
}

    --bg-items: #1a1b2e;

    --bg-selected: #3d3e5a;
    --text-primary: #e4e6ed;

    --border-color: #2e3143;
    --border-focus: #818cf8;

// Aplicar tema personalizado
const search = new BuscadorDinamico.Search({
    element: '#app',
    data: items,
    theme: 'mi-tema'
}).init();

Soporte dark/light mode

El tema adaptative (tema por defecto) detecta automáticamente el modo del sistema usando selectores CSS con el atributo data-theme:

/* El tema adaptative usa selectores de atributo */
[data-theme="light"] .app-search.theme-adaptative {
    --search-bg-color: #ffffff;
    --search-text-color: #111827;
    /* ... modo claro */
}

[data-theme="dark"] .app-search.theme-adaptative {
    --search-bg-color: #161b22;
    --search-text-color: #e6edf3;
    /* ... modo oscuro */
}

Para que funcione, establece el atributo data-theme en el elemento <html>:

<!-- Modo claro -->
<html lang="es" data-theme="light">

<!-- Modo oscuro -->
<html lang="es" data-theme="dark">
💡
Consejo

El tema adaptative es el tema por defecto y está completamente funcional con soporte para light/dark mode. No requiere configuración adicional más allá de establecer el atributo data-theme en el HTML.
