Instalación

Múltiples formas de instalar el componente según tu entorno de desarrollo.

Vía npm / pnpm

La forma recomendada para proyectos con bundler (Vite, Webpack, Rollup, etc.):

# Con npm
npm install buscador-dinamico

# Con pnpm (recomendado)
pnpm add buscador-dinamico

Luego impórtalo en tu JavaScript:

import { Search } from 'buscador-dinamico';
import 'buscador-dinamico/css/buscador-dinamico.css';

Vía CDN (UMD)

Para proyectos sin bundler o inclusiones rápidas:

<!-- CSS -->
<link rel="stylesheet" href="https://unpkg.com/buscador-dinamico/dist/css/buscador-dinamico.css">

<!-- JavaScript (UMD) -->
<script src="https://unpkg.com/buscador-dinamico/dist/buscador-dinamico.umd.js"></script>

<script>
  const search = new BuscadorDinamico.Search({
    element: '#my-search',
    data: [...]
  }).init();
</script>

Vía script tag (ES Module)

<link rel="stylesheet" href="./dist/css/buscador-dinamico.css">

<script type="module">
  import { Search } from './dist/buscador-dinamico.es.js';

  const search = new Search({
    element: '#my-search',
    data: [...]
  }).init();
</script>

Descarga manual

Puedes descargar los archivos directamente desde la sección de releases de GitHub:

  1. Descarga buscador-dinamico.es.js (módulo ES) o buscador-dinamico.umd.js (UMD)
  2. Descarga buscador-dinamico.css desde dist/css/
  3. Inclúyelos en tu proyecto

Estructura de archivos

Después de la instalación, tu proyecto debería tener:

tu-proyecto/
├── node_modules/
│   └── buscador-dinamico/
│       ├── dist/
│       │   ├── buscador-dinamico.es.js
│       │   ├── buscador-dinamico.umd.js
│       │   └── css/
│       │       └── buscador-dinamico.css
│       └── src/
│           └── css/ (estilos fuente)
💡
Con Vite

Vite detecta automáticamente los archivos CSS importados y los incluye en el bundle final.

Verificación

Para verificar que la instalación fue exitosa, abre la consola del navegador y ejecuta:

console.log(typeof BuscadorDinamico.Search); // "function" (UMD)
// o
import { Search } from 'buscador-dinamico';
console.log(typeof Search); // "function" (ES Module)
