API

Referencia completa de la interfaz pública del componente.

Constructor

new Search(params: SearchParams): Search

Crea una nueva instancia del componente. Ver Configuración para los parámetros disponibles.

Métodos públicos

init()

init(): Search

Inicializa el componente, renderiza la estructura DOM y configura los event listeners. Debe llamarse después del constructor.

const search = new Search({ element: '#app', data: items }).init();

draw(searchTerm?, isEvent?)

draw(searchTerm?: string, isEvent?: boolean): Promise<Search>

Ejecuta una búsqueda y renderiza los resultados. Se llama automáticamente al escribir en el input.

Parámetro Tipo Default Descripción
searchTerm string this.searchTerm Término de búsqueda
isEvent boolean false Si fue iniciado por evento del usuario

sort(field, order?)

sort(field: string, order?: 'asc' | 'desc'): Search

Ordena los datos por un campo específico. En modo servidor, ejecuta una nueva búsqueda.

search.sort('nombre', 'asc');
search.sort('precio', 'desc');

clearSort()

clearSort(): Search

Elimina el orden actual y reinicia al orden natural de los datos.

showLoading()

showLoading(): Search

Muestra un indicador de carga en el contenedor de resultados.

on(eventName, callback)

on(eventName: string, callback: Function): EventEmitter

Registra un listener para un evento. Retorna el EventEmitter para encadenamiento.

search
    .on('search', (data) => console.log('Búsqueda:', data))
    .on('pageChange', (data) => console.log('Página:', data.page))
    .on('itemSelected', (data) => console.log('Seleccionado:', data.item));

getCacheKey(searchTerm, page)

getCacheKey(searchTerm: string, page: number): string

Genera la clave de caché para una búsqueda y página específicas.

setupKeyboardNavigation()

setupKeyboardNavigation(): Search

Configura la navegación por teclado (flechas arriba/abajo, Enter para seleccionar).

destroy()

destroy(): void

Destruye la instancia, limpiando todos los event listeners, observer de scroll y referencias internas.

search.destroy(); // Limpia todo

Propiedades públicas

Propiedad Tipo Descripción
element string Selector CSS del contenedor
searchTerm string Término de búsqueda actual
data Array Array de datos originales
theme string Tema CSS activo
pagination Pagination Instancia del módulo de paginación
events EventEmitter Instancia del sistema de eventos
cache LRUCache Instancia del sistema de caché
renderer SearchRenderer Instancia del motor de renderizado

Encadenamiento

La mayoría de métodos retornan this para permitir encadenamiento:

search
    .sort('nombre', 'asc')
    .draw('laptop')
    .then(() => console.log('Búsqueda completada'));
