Eventos

El componente emite eventos personalizados que permiten reaccionar a diferentes acciones del usuario y del sistema. Todos los eventos están tipados mediante SearchEventMap para TypeScript.

Uso básico

const search = new BuscadorDinamico.Search({
    element: '#app',
    data: items
}).init();

search.on('search', (data) => {
    console.log('Término de búsqueda:', data.searchTerm);
    console.log('Resultados encontrados:', data.totalResults);
});

search.on('itemSelected', (data) => {
    console.log('Item:', data.item);
    data.close(); // Cierra el panel de resultados
});

Lista de eventos

init

Se emite cuando el componente termina de inicializarse.

// Tipo del dato emitido
interface SearchEventInit {
    searchTerm: string;
    itemsPerPage: number;
    procesServer: boolean;
}
search.on('init', (data) => {
    console.log('Componente inicializado:', data);
});

search

Se emite después de cada búsqueda ejecutada (solo si fue iniciada por evento del usuario).

interface SearchEventData {
    searchTerm: string;
    results: Record<string, any>;
    totalResults: number;
    timestamp: string; // ISO date
}

pageChange

Se emite cuando cambia la página actual (scroll infinito).

interface PageChangeEventData {
    page: number;
    totalPages: number;
    itemsOnPage: number;
    totalLoaded: number;
}
search.on('pageChange', (data) => {
    console.log(`Página ${data.page} de ${data.totalPages}`);
    console.log(`Items cargados: ${data.totalLoaded}`);
});

resultsCleared

Se emite cuando cambia el término de búsqueda y se limpian los resultados anteriores.

interface ResultsClearedEventData {
    previousSearchTerm: string;
}
search.on('resultsCleared', (data) => {
    console.log('Término anterior:', data.previousSearchTerm);
});

searchComplete

Se emite cuando la búsqueda completa renderiza sus resultados.

interface SearchCompleteEventData {
    searchTerm: string;
    results: Record<string, any>[];
    totalResults: number;
}
search.on('searchComplete', (data) => {
    console.log('Búsqueda:', data.searchTerm);
    console.log('Total:', data.totalResults);
});

sortChange

Se emite cuando cambia el ordenamiento de los datos.

interface SortChangeEventData {
    field: string;
    order: 'asc' | 'desc';
}

itemSelected

Se emite cuando el usuario selecciona un item (clic o Enter).

interface ItemSelectedEventData {
    item: Record<string, any>;
    index: number;
    close: () => void; // Función para cerrar los resultados
}
search.on('itemSelected', ({ item, index, close }) => {
    console.log('Item seleccionado:', item);
    console.log('Índice:', index);
    close(); // Cierra el panel de resultados
});

itemHighlighted

Se emite cuando un item recibe foco visual (navegación por teclado o clic).

interface ItemHighlightedEventData {
    item: Record<string, any>;
    index: number;
}

appendItems

Se emite cuando se añaden nuevos items al DOM.

interface AppendItemsEventData {
    items: any[];
    content: HTMLElement; // Contenedor de items
}

destroy

Se emite justo antes de destruir la instancia.

interface DestroyEventData {
    timestamp: string;
}

error

Se emite cuando ocurre un error durante la búsqueda o inicialización.

interface ErrorData {
    code: string;      // Código de error (ej: "SEARCH_020")
    message: string;   // Mensaje descriptivo
    solution: string;  // Guía para resolver el error
    context: any;      // Contexto adicional del error
}
search.on('error', (data) => {
    console.error('Código:', data.code);
    console.error('Mensaje:', data.message);
    console.error('Solución:', data.solution);
});

Remover listeners

// Remover un listener específico por referencia
const callback = (data) => console.log(data);
search.on('search', callback);
search.events.off('search', callback);

// Remover todos los listeners de un evento
search.events.removeAllListeners('search');

// Remover todos los listeners de todos los eventos
search.events.removeAllListeners();

Una sola vez

// Se ejecuta solo la primera vez que se emite
search.events.once('init', (data) => {
    console.log('Inicialización (solo una vez):', data);
});

Eventos asíncronos

// Emitir evento de forma asíncrona
await search.events.emitAsync('search', {
    searchTerm: 'laptop',
    results: data,
    totalResults: 10,
    timestamp: new Date().toISOString()
});
// Promise.allSettled internamente — errores individuales no bloquean

Información de listeners

// Cantidad de listeners de un evento
const count = search.events.listenerCount('search');

// Nombres de todos los eventos registrados
const names = search.events.eventNames();

// Lista detallada de eventos con cantidad de listeners
const list = search.events.getEventList();
// [{ event: 'search', listeners: 2 }, { event: 'init', listeners: 1 }]

SearchEventMap (TypeScript)

Todos los eventos están tipados mediante la interfaz SearchEventMap, que asocia cada nombre de evento con su tipo de datos:

interface SearchEventMap {
    init: SearchEventInit;
    search: SearchEventData;
    searchComplete: Omit<SearchEventData, 'timestamp'>;
    resultsCleared: ResultsClearedEventData;
    pageChange: PageChangeEventData;
    sortChange: SortChangeEventData;
    itemHighlighted: ItemHighlightedEventData;
    itemSelected: ItemSelectedEventData;
    appendItems: AppendItemsEventData;
    destroy: DestroyEventData;
    error: ErrorData;
}

// El tipo del callback se infiere automáticamente:
search.on('itemSelected', (data) => {
    // data: ItemSelectedEventData (inferido)
    console.log(data.item, data.index);
    data.close();
});
