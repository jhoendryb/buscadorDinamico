Eventos

El componente emite eventos personalizados que permiten reaccionar a diferentes acciones del usuario y del sistema.

Uso básico

const search = new BuscadorDinamico.Search({
    element: '#app',
    data: items
}).init();

search.on('search', (data) => {
    console.log('Término de búsqueda:', data.searchTerm);
    console.log('Resultados encontrados:', data.results.length);
});

Lista de eventos

init

Se emite cuando el componente termina de inicializarse.

// Tipo del dato emitido
interface SearchEventInit {
    searchTerm: string;
    itemsPerPage: number;
    procesServer: boolean;
}
search.on('init', (data) => {
    console.log('Componente inicializado:', data);
});

search

Se emite después de cada búsqueda ejecutada.

interface SearchEventData {
    searchTerm: string;
    results: Record<string, any>[];
    count: number;
}

pageChange

Se emite cuando cambia la página actual (scroll infinito o paginación manual).

interface PageChangeEventData {
    page: number;
    totalPages: number;
    itemsOnPage: number;
    totalLoaded: number;
}
search.on('pageChange', (data) => {
    console.log(`Página ${data.page} de ${data.totalPages}`);
    console.log(`Items cargados: ${data.totalLoaded}`);
});

sortChange

Se emite cuando cambia el ordenamiento de los datos.

interface SortChangeEventData {
    field: string;
    order: 'asc' | 'desc';
}

itemSelected

Se emite cuando el usuario selecciona un item (clic o Enter).

interface ItemSelectedEventData {
    item: Record<string, any>;
    index: number;
    close: () => void; // Función para cerrar los resultados
}
search.on('itemSelected', ({ item, index, close }) => {
    console.log('Item seleccionado:', item);
    console.log('Índice:', index);
    close(); // Cierra el panel de resultados
});

itemHighlighted

Se emite cuando un item recibe foco visual (navegación por teclado o clic).

interface ItemHighlightedEventData {
    item: Record<string, any>;
    index: number;
}

appendItems

Se emite cuando se añaden nuevos items al DOM (scroll infinito).

interface AppendItemsEventData {
    items: Record<string, any>[];
    isFirstPage: boolean;
}

destroy

Se emite justo antes de destruir la instancia.

interface DestroyEventData {
    timestamp: string;
}

error

Se emite cuando ocurre un error durante la búsqueda o inicialización.

search.on('error', (error) => {
    console.error('Error en el componente:', error);
});

Remover listeners

// Remover un listener específico
const callback = (data) => console.log(data);
search.on('search', callback);
search.events.off('search', callback);

// Remover todos los listeners de un evento
search.events.removeAllListeners('search');

// Remover todos los listeners
search.events.removeAllListeners();

Una sola vez

// Se ejecuta solo la primera vez que se emite
search.events.once('init', (data) => {
    console.log('Inicialización (solo una vez):', data);
});
