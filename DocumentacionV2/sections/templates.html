Plantillas

Personaliza la apariencia de cada item de resultado con plantillas de string o funciones JavaScript.

Plantilla de string

Usa comillas simples o backticks para definir HTML con interpolación simple:

const search = new BuscadorDinamico.Search({
    element: '#app',
    data: productos,
    template: '<div class="producto"><strong>{nombre}</strong> - ${precio}</div>'
}).init();
💡
Sintaxis de interpolación

Usa la sintaxis de doble llave {{campo}} para insertar valores del item. El sistema reemplaza automáticamente estas marcas con los valores correspondientes.

Plantilla con función

Para mayor control, usa una función que reciba el item y un callback de resaltado, y retorne HTML:

const search = new BuscadorDinamico.Search({
    element: '#app',
    data: productos,
    template: (item, highlightText) => `
        <div class="card">
            <img src="${item.imagen}" alt="${item.nombre}">
            <h3>${highlightText(item.nombre)}</h3>
            <p class="price">$${item.precio.toFixed(2)}</p>
            <span class="category">${item.categoria}</span>
        </div>
    `
}).init();
💡
Callback highlightText

Cuando highlightEnabled está activo, el segundo parámetro highlightText es una función que envuelve el texto con un <span> resaltado. Puedes usarla en cualquier campo del item. Si highlightEnabled es false, el callback retorna el texto sin modificar.

Ejemplo práctico

Creando una tarjeta de producto completa con resaltado:

const template = (item, highlightText) => `
    <div class="product-card" data-id="${item.id}">
        <div class="product-card__image">
            <img src="${item.image}" alt="${item.name}" loading="lazy">
        </div>
        <div class="product-card__body">
            <h3 class="product-card__title">${highlightText(item.name)}</h3>
            <p class="product-card__desc">${highlightText(item.description)}</p>
            <div class="product-card__footer">
                <span class="product-card__price">$${item.price}</span>
                <span class="product-card__badge">${item.category}</span>
            </div>
        </div>
    </div>
`;

Plantilla por defecto

Si no se proporciona una plantilla, el componente usa la siguiente estructura:

<div class="items">
    <span class="item-name">{valor del campo}</span>
</div>
⚠️
XSS

Si usas innerHTML directamente en plantillas con datos del usuario, asegúrate de sanitizar el contenido para prevenir ataques de inyección de scripts.

Acceso a datos del item

Dentro de la plantilla, tienes acceso a todas las propiedades del objeto item:

// Datos de ejemplo
const data = [
    { id: 1, name: 'Laptop', price: 999.99, category: 'Electrónica', active: true }
];

// Plantilla accediendo a todas las propiedades
template: (item) => `
    <div>
        <h3>${item.name}</h3>
        <p>Precio: $${item.price}</p>
        <p>Categoría: ${item.category}</p>
        <p>Estado: ${item.active ? 'Activo' : 'Inactivo'}</p>
    </div>
`
